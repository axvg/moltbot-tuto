Security Checklist

Mandatory before production deployment.

Non-Negotiable Requirements

UFW firewall enabled. SSH key auth only (no passwords). API tokens with 600 permissions. Port 3000 IP-restricted. Fail2Ban installed.

UFW Firewall Rules
# Install & configure apt install ufw -y ufw default deny incoming ufw default allow outgoing # Allow specific IPs only ufw allow from YOUR_OFFICE_IP to any port 22 ufw allow from YOUR_HOME_IP to any port 3000 # Block dangerous ports ufw deny 3306/tcp # MySQL ufw deny 5432/tcp # PostgreSQL ufw deny 6379/tcp # Redis ufw deny 27017/tcp # MongoDB ufw enable
